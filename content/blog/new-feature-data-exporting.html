New Feature: Data Exporting

By Josh Pigford on March 20, 2014 Last updated on April 23, 2026

style
article-body

This week turned out to be much more exciting than initially planned, as far as features go.

Notifications, Revenue Churn, Refunds and now another frequently requested feature: Data Exporting!

You can now export the chart data on all of your metrics. Just set the date range of data you want, then export and you’ll get the same dates and numbers we use to generate the line graphs.

This is definitely a simplistic first go at this. There’s a lot more we’ve got in the works to let you export the data we generate, so if there’s something specific you’d like to be able to export, please let me know.

Exporting is available to all users on all plans.
